Installation Keys

The installation key is used to add resources to the iWe Network.
  • Download an installation key. The installation key will expire after {{ expire_hours }} hours, but the same key can be used for multiple installarions.
  • Copy the installation key to the installer media(probably a USB stick)
  • Boot the installer on the server you want to add to iWe
  • When the installarion starts, you will get a notification to activate the new server
  • Activate your new server {% endblock %}
